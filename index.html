Pen to Code: Emulating Shakespeare’s Prose through Style Transfer

Developed a Copy-Enriched Sequence-to-Sequence (Seq2Seq) NLP model using Python and PyTorch to translate modern English into Shakespearean prose. Leveraged advanced AI models, including Transformer architectures like BERT and GPT, to capture linguistic nuances and stylistic elements of Shakespearean writing. Focused on data preprocessing, hyperparameter tuning, and linguistic style accuracy, achieving 94% fidelity in emulating Shakespeare’s prose. Delivered a robust, end-to-end solution that combines AI-driven style transfer with interpretable language generation for creative and educational applications.

Analyzing Cricket: Shot Recognition & Similarity

In the CricShot Analysis project, advanced Artificial Intelligence is employed using a sophisticated model based on the EfficientNet framework to classify cricket shot videos into distinct categories with a remarkable 94% accuracy. The model parameters are optimized using a Genetic Algorithm, enhancing player performance analysis and strategic game planning. This AI-driven approach culminates in a user-friendly web application, facilitating real-time video classification and similarity comparisons, significantly impacting coaching strategies and enhancing the sport’s competitive dynamics.
